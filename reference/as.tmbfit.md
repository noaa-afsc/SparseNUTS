# Construtor for tmbfit objects

Construtor for tmbfit objects

## Usage

``` r
as.tmbfit(x, parnames, mle, invf, metric, model = "anonymous")
```

## Arguments

- x:

  A fitted MCMC object

- parnames:

  A character vector of unique par names

- mle:

  A list of MLE parameters

- invf:

  The inverse function for the parameters

- metric:

  The metric used

- model:

  A character giving the model name
