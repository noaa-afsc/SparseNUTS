# Get the joint covariance Sigma from an optimized TMB or RTMB obj without random effects.

Get the joint covariance Sigma from an optimized TMB or RTMB obj without
random effects.

## Usage

``` r
.get_Qinv(obj)
```

## Arguments

- obj:

  An optimized TMB or RTMB object

## Value

A dense matrix Sigma
